# ✅ To Do List App — Day 1 of #30Days30Projects

A clean, minimal, and fully functional To-Do List application
built with pure HTML, CSS, and Vanilla JavaScript. Designed with
a modern gradient background and a smooth card-based UI that
makes task management simple, fast, and visually pleasing.

## ✨ Features
- ➕ Add new tasks instantly with the Add button
- ✅ Mark tasks as completed with a single click
- ❌ Delete tasks with the close button
- 🎨 Beautiful gradient background — blue to purple
- 🖊️ Clean card UI with smooth interactions
- ⚡ Built with zero dependencies — pure Vanilla JavaScript

## 🛠️ Tech Stack
- HTML5
- CSS3 — Custom gradient, card design & styling
- JavaScript (Vanilla) — DOM manipulation & event handling
